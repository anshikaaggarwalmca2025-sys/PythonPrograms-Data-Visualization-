# Power BI Visualization Lab

## Objective
Transforming raw data into interactive and meaningful visual stories to drive data-based decisions.

## Dashboard Features
* **Data Modeling:** Establishing relationships between different data sources.
* **Visual Gallery:** Use of Bar Charts, Slicers, Map Visuals, and KPI Cards.
* **Power Query:** Cleaning, transforming, and "mashing up" data before visualization.
* **Interactive Elements:** Implementing cross-filtering and drill-through actions.

## Learning Outcome
- Ability to create end-to-end business intelligence reports.
- Understanding the importance of UI/UX in data storytelling.
