# SQL Laboratory Work

## Objective
To master Relational Database Management System (RDBMS) concepts through practical implementation of SQL queries.

## Key Concepts Covered
* **Schema Definition:** Designing tables with primary keys, foreign keys, and constraints (NOT NULL, UNIQUE, CHECK).
* **Advanced Querying:** Mastering `GROUP BY`, `HAVING`, and `ORDER BY` clauses for data aggregation.
* **Joins:** Practical implementation of `INNER JOIN`, `LEFT JOIN`, and `RIGHT JOIN` to combine multiple tables.
* **Subqueries:** Writing nested queries to solve complex data retrieval problems.

## Tools Used
* MySQL / Oracle / PostgreSQL (Mention whichever you use in college)
