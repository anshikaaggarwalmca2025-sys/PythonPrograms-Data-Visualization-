# Python Programming Lab

## Objective
Building a strong foundation in Python programming, focusing on logic development and automation scripts.

## Topics Covered
* **Core Logic:** Implementation of decision-making statements and efficient looping.
* **Functional Programming:** Creating reusable code blocks and understanding scope.
* **Data Handling:** Using Pandas and NumPy for basic data manipulation and analysis.
* **Exception Handling:** Managing errors gracefully using `try-except` blocks.
* **OOP Concepts:** Basic understanding of Classes and Objects within the lab tasks.

## Tools Used
* Python 3.x
* VS Code / Jupyter Notebook
